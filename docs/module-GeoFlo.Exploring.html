    GeoFlo.Exploring

    A class that handles exploring functionality in a mapping context. Only enabled when the currentMode is set to 'draw'.

    Methods

    static activate() → {void}
    src/action/Exploring.js, line 12

    Activates by setting the 'enabled' property to true and enabling the 'exploring' option.
    Returns:
    void

    static deactivate()
    src/action/Exploring.js, line 25

    This function sets the 'enabled' and 'currentMatch' properties to false, and disables the 'exploring' option in the context.

    static getMatch(coords, options) → {Promise.<(Array|Boolean)>}
    src/action/Exploring.js, line 124

    This function takes an array of coordinates and optional options to fetch a matched route from the OSRM API.
    Parameters:
    Name Type Description
    coords Array An array of coordinates to match the route against.
    options Object Optional parameters for customization.
    Properties
    Name Type Description
    feature Object Optional feature to match against.
    radius Number | function Optional radius for matching or a function to calculate it.
    profile String Optional profile for routing (default: 'driving').
    set Boolean Optional flag to set the feature if matching is successful.
    Returns:
    Promise.<(Array|Boolean)> - The matched route response object or false if no coordinates are provided or matching fails.
